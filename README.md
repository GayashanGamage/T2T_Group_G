## Initialize Local Repository & Push to GitHub

1. Open a terminal and run:

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/T2T_Group_G.git
cd T2T_Group_G
```

2. Initialize Git:

```bash
git init
```

3. Add files:

```bash
git add .
```

4. Commit changes:

```bash
git commit -m "Initial commit"
```
